Project start

Once you check out the project make sure you execute the following commands

Make sure Node is installed.
https://nodejs.org/en/download/

After Node is installed go to the project directory and execute

$ npm install

If you have previously installed a version of gulp globally,
please run
$ npm rm --global gulp
to make sure your old version doesn't collide with gulp-cli.

$ npm install --global gulp-cli

$ bower install

after installing run
$ gulp
in the directory of gulpfile.js
